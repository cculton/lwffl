How this site works
Both the H2H app and the Career Leaders page read from a single JSON file:
league-scores.json
  • One entry per matchup (year, week, managers, scores, playoffs flag).
  • Career stats, streaks, and weekly highs are computed on the fly.
  • No manual recalcs ‚Äì just update the JSON and redeploy.
Keeping it updated
Typical weekly workflow:
  • 1. Add new games to league-scores.json.
  • 2. Commit & push to the GitHub repo.
  • 3. Netlify (or your host) auto-deploys the new stats.

Need a new view or record type later? It can be computed from the same JSON ‚Äì no new data format required.
